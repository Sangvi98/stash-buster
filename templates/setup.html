Setup Required

Stash Buster needs two sets of Ravelry credentials. Follow these steps:

1. Create an OAuth 2.0 App (for login)

Go to ravelry.com/pro/developer and create an app with OAuth 2.0 access.

Set the callback URL to: https://localhost:5001/callback

2. Create a Basic Auth App (for data)

Create a second app with "Basic Auth: read only" access type. This provides the API keys for reading stash and project data.

3. Create a .env file

Copy .env.example to .env and fill in all four keys:

cp .env.example .env

4. Restart the server

Stop the server (Ctrl+C) and re-run python3 app.py
